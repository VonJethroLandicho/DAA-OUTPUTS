Stock GDP vs Population Plot

Displays a scatter plot showing correlation between population and GDP growth.

Description

This script generates a scatter plot comparing GDP growth rate against population growth from 2014–2023 using hardcoded values. It calculates and displays the correlation coefficient and draws a trendline using linear regression.

How It Works

The program performs the following steps:

  1. Loads Hardcoded Data: Uses predefined values for years, GDP growth, and population growth
  2. Generates Scatter Plot: Plots GDP vs population growth with year labels
  3. Calculates Correlation: Computes Pearson correlation coefficient
  4. Adds Trendline: Uses linear regression to show overall direction

Execution Details

  • Data Source: None (hardcoded)
  • Years Covered: 2014 – 2023
  • GDP Growth Values: [2.1, 2.4, 2.5, 2.6, 2.9, 3.0, 2.7, 2.8, 3.1, 3.3]
  • Population Growth: [1.0, 1.1, 1.2, 1.3, 1.4, 1.5, 1.6, 1.7, 1.8, 1.9]
  • Correlation: ~0.94 (strong positive correlation)
