Bubble Sort

Dataset: Updated_Mobile_Dataset.csv

This script reads a CSV file containing mobile phone data, validates its contents, and performs Bubble Sort on the 'Price' column based on user preference (ascending or descending).

Key Features

  • Data validation before sorting
  • Skips invalid rows and reports them
  • Saves sorted result to a new CSV file

Flip Booleans

Dataset: machine_problems_dataset.csv

This script reads from a CSV file and flips boolean values ('True' <-> 'False') in the fourth column after validating that the dataset is properly formatted.

Key Features

  • Checks for missing or extra columns
  • Reports invalid boolean values
  • Displays modified dataset and change count

Convex Hull Chatbot [Access files for testing]

This program is a Flask-based chatbot web application that uses a custom LLaMA 3 model‚Äîcalled "convexhullmodel"‚Äîto answer questions specifically about the convex hull algorithm.

üîß Backend Components

  1. Flask App
    Provides a simple web server with two routes:
    • /: Serves the main web page (likely index.html).
    • /chat: Accepts POST requests for user messages and returns model responses.
  2. Custom Language Model
    OllamaLLM(model="convexhullmodel") uses Langchain with Ollama to load and query a local LLaMA 3 model. This model focuses on convex hull concepts, including:
    • Definitions
    • Algorithms (Graham's Scan, Gift Wrapping, QuickHull)
    • Historical context
    • Applications
  3. Conversation Handling
    Uses an in-memory conversation_history list (tracks last 20 messages). Sends full context as a prompt for coherent replies.

üí¨ Chatbot Behavior

The system prompt defines behavior:

  • Friendly, informative, convex hull‚Äìfocused.
  • Deflects unrelated questions:
    • Personal info ‚Üí Says it has none.
    • General algorithms ‚Üí Says it only covers convex hulls.
    • Emotions ‚Üí Says it cannot feel.
    • Off-topic/spam ‚Üí Redirects politely.
