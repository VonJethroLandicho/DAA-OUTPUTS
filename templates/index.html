Bubble Sort

Dataset: Updated_Mobile_Dataset.csv

This script performs a Bubble Sort on mobile phone pricing data based on user preference (ascending/descending).

Key Features

  • Data validation before sorting
  • Skips invalid rows and reports them
  • Saves sorted result to a new CSV file

Example Input Prompt


Would you like to sort by price?
Enter 'A' for ascending or 'D' for descending order: A

Example Output


Sorted dataset saved to Datasets/Sorted_Mobile_Dataset_Ascending.csv
Sorting completed in 0:05 (5 seconds)

Flip Booleans

Dataset: machine_problems_dataset.csv

This script flips boolean values ('True' <-> 'False') in the fourth column after validating that the dataset is properly formatted.

Key Features

  • Checks for missing or extra columns
  • Reports invalid boolean values
  • Displays modified dataset and change count

Example Input Dataset


ID,Problem,Status,Solved
1,Sorting,Easy,True
2,Searching,Medium,False
3,Hashing,Hard,True

Example Output


Modified Dataset:
['ID', 'Problem', 'Status', 'Solved']
['1', 'Sorting', 'Easy', 'False']
['2', 'Searching', 'Medium', 'True']

Total number of changes: 1

Stock GDP vs Population Plot

Dataset: None (hardcoded values)

Generates a scatter plot showing correlation between population and GDP growth from 2014–2023 using hardcoded values. It also calculates and draws a trendline using linear regression.

Key Features

  • Displays correlation coefficient
  • Annotates each point with year and GDP value
  • Draws a trendline using linear regression

Example Hardcoded Data


Years = [2014, 2015, 2016, 2017, 2018, 2019, 2020, 2021, 2022, 2023]
GDP_growth = [2.1, 2.4, 2.5, 2.6, 2.9, 3.0, 2.7, 2.8, 3.1, 3.3]
Population_growth = [1.0, 1.1, 1.2, 1.3, 1.4, 1.5, 1.6, 1.7, 1.8, 1.9]

Example Output

Graph shows GDP growth rate vs population growth with a correlation coefficient and trendline.

Subset Sum Group Selector

Dataset: Students_Grading_Dataset.csv

Finds a subset of students matching score, study hours, and stress level criteria using backtracking.

Key Features

  • Allows user-defined constraints
  • Selects up to 50 students
  • Displays results with bar graphs and tables

Example User Input


Enter minimum total score: 70
Enter maximum total score: 90
Enter minimum study hours per week: 10
Enter maximum study hours per week: 20
Enter maximum stress level (1-10): 5
Enter the number of students required for the group (Maximum 50): 5

Example Output

Bar graph showing selected students’ scores, study hours, and stress levels + table with student details
