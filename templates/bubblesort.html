Description

This script reads a CSV file containing mobile phone data, validates its contents, and performs Bubble Sort on the 'Price' column using predefined settings.

How It Works

The program follows these steps:

  1. Loads Data: Reads the CSV file: Updated_Mobile_Dataset.csv
  2. Validates Data: Checks for missing values and invalid price entries
  3. Sorts by Price: Uses Bubble Sort algorithm in ascending order (predefined)
  4. Saves Output: Writes the sorted results into a new CSV file: Sorted_Mobile_Dataset_Ascending.csv

Execution Details

  • Algorithm: Bubble Sort
  • Sorting Criteria: Price (Ascending)
  • Input File: Updated_Mobile_Dataset.csv
  • Output File: Sorted_Mobile_Dataset_Ascending.csv
  • Dataset Size: First 500 rows used for sorting
  • Time Complexity: O(n²) — suitable for small datasets like this one
